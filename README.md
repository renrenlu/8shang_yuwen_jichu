# 语文基础知识互动练习

面向初中语文基础知识复习的互动题库，共收录 8 个板块、852 道题。

## 在线使用

[直接打开互动练习册](https://renrenlu.github.io/8shang_yuwen_jichu/)

## 功能

- 每个板块每次随机抽取 20 题，完整覆盖一轮前不重复。
- 综合练习按各板块题量同比例抽取，每次 30 题。
- 错题自动写入错题本，答对后自动移出。
- 原题扫描、PDF 原页和参考答案页可随时对照，并支持翻页。
- 可通过 JSON 模板继续导入新的资料板块。
- 学习进度保存在当前设备的浏览器中。

## 本地运行

需要 Node.js 22.13 或更高版本。

```bash
npm install
npm run dev
```

完整检查：

```bash
npm run lint
npm test
```

新增板块时，可下载网站页脚中的题库模板并按相同结构填写后导入。
